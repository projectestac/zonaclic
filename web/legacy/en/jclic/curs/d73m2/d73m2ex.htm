Module 2
The Creation of Educational Activities using JClic
Exercises
The creation of activities: puzzle, memory game and wordsearch.
   

Inside a new project, create three activities with the following characteristics:

  • A puzzle of the type you prefer.
  • A memory game where the pairs are 6 images and there is a text related to each one of these images (could be the names of the objects represented in the images).
  • A wordsearch with 6 words, which could be the ones used in the memory game and the images used as relevant material.

In each activitty the aspect of the windows and the messages must be worked on.

The three activities must have the same theme.

The images can be obtained from the Internet.

In the Xtec store of Catalan Icons you can find drawings in GIF format grouped into themes and of the correct size for the activities memory and wordsearch.

Photographs, also divided into themes but of a larger size can be found in the Image Bank MEC

For the puzzle you can look for a larger image suitable for the theme you choose in the Google Image search engine.

You will find what you are looking for more easily from the Advanced Image Search, indicating for example that you want images of a medium size. This of course, if you want a picture or photo or drawing for your puzzle.

When you finish the exercise save it and send it to your tutor.

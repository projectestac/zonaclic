Information Screen
   

This is a single grid that has a piece of related information that can be the beginning of a project, or a multimedia content.

It is used in many cases to show certain information which the user must read, look at or listen to before continuing the rest of the activities.

It is recommended that JClic projects start with an information screen. In only one box you can put the title and the author, and illustrate the content of the project with pictures. You can also begin with an information screen that has different boxes which refer to different levels or routes.

You can also find projects that link an initial information screen containing the title page and credits to a second screen with a choice of activity sequences. In the design for the sequences, the final activity links up again with the initial screen so that the user can choose a different sequence.

From JClic author, the design options are the same as for the rest of the activities in this module.

Text: order elements

   
In this practice session putting elements in order, as in the previous one, you will only practice one (putting words in order) of the two possibilities that exist. (putting paragraphs in order and putting words in order). The way of working is similar, but instead of selecting a complete paragraph to put in order, you simply choose words.
   

Open the file module4.jclic.zip from JClic author. Create a new text activity Text: order elements. Give the same name to the activity: order elements. Recover the text which you inserted into the activity Fill-in blanks writing in a blank space (The Merchant of Venice). Remember to use the combination Control+C to copy and Control+V to paste.

The same as in the identify elements activities, the button is activated so as you can choose to put either paragraphs or words in order. Make sure it looks like this:

Choose the following words as unknown elements: summer, winds, buds, heaven, complexion, death, life. All the chosen words are nouns so as to be able to interchange them in the text. Remember that you have to select the word and click on .

Change the style of the main text and also the unknown elements (using the button ). Resize the body of the text so that it can be read without difficulty.

Test the activity works.

If this is the first time that this type of activity is being done, it is difficult to make sense of. For this reason, it would be better to show the correct version of the text before beginning the activity. If you indicate in JClic that the correct version should be shown before starting, the pupil will see the words to be mixed up in a different colour. This could make the activity too easy. To show the text without distinguishing marks you must do the following:

Select the complete text and copy it using Ctrl+C.

Click on the button . In the dialogue box select Display this text. Place the cursor on the box under this option and paste the text using Ctrl+V.

Decide on a style that you like for the text. It´s better if the font size is the same as the one you have chosen for the exercise.

Write a message telling the pupils to read the text:

Then write an initial message giving the instructions to put the words in the text in their correct places, and a final message congratulating the pupils for doing the activity correctly.

Go to button . The button Evaluation is predefined and activated.Change the sentence:

Press and do the activity. If there is anything you don´t like with regards to the style, modify it before saving the project. module4.jclic.zip.
 
   

To make an activity where paragraphs have to be put in order, you must choose a text where the paragraphs are clearly delimited. e.g. you can use a list of instructions, short stories where the sequence is clear, poems which are known to the pupils...

Text: identify elements
   
In this practice session you will create the identify words activity using a sonnet. The pupil will work on words which you have previously selected. You could also choose specific characteristics from the text, which would have to be identified later. These could be activities on looking for the stress in different words, consonants with a certain sound...
   

Open the project module4.jclic.zip. Create a new text activity for Text: identify elements. Call it : Identify words.

From Text, copy this sonnet by William Shakespeare:

XVIII.

Shall I compare thee to a summer's day?
Thou art more lovely and more temperate:
Rough winds do shake the darling buds of May,
And summer's lease hath all too short a date:
Sometime too hot the eye of heaven shines,
And often is his gold complexion dimm'd;
And every fair from fair sometime declines,
By chance, or nature's changing course un-
trimm'd;
But thy eternal summer shall not fade,
Nor lose possession of that fair thou ow'st,
Nor shall death brag thou wander'st in his
shade,
When in eternal lines to time thou grow'st;
So long as men can breathe, or eyes can see,
So long lives this, and this gives life to thee.

You can see that by choosing this type of activity, the button type of activity has been activated. In the gap filling activities this button was not marked.

Click on the button and verify that the option Identify words has been selected.

Mark all the adjectives in the sonnet as unknown elements. You do this by selecting the word and clicking on the button . No dialogue box will open. When you have marked all the adjectives, test the activity using .

The initial message you write will have to explain that all the adjectives have to be marked so as the pupil knows what he/she has to do in the activity.

Return to the activity and write the initial message and the final message in which you will congratulate the pupil on finding all the adjectives.

From Text, select the button Check . Change the text which appears (Check) to Evaluate yourself. Remember that the background colour for the evaluation button will be the same as the game window.

Style the main text and the unknown elements. Each time the pupil marks an adjective, this must be sufficiently visible to be recognised. It is important that the errors are clear after pressing the evaluation bar.

Resize the main window.

Test that the activity works properly and save the project. module4.jclic.zip.
 
   

You should explain to the pupils what the correct and incorrect unknown elements look like (visual y auditive).

The sounds of events tab is activated automatically in Options | User interface. When there is an error, the sound given off is obvious. Depending on the level of the activity and if you are interested in doing so, the tab can be deactivated. You can also deactivate from the tab Project | User interface if you do not wish any sound in the whole project.

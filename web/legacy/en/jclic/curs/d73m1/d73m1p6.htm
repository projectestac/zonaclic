The messages of an activity.
 
The objective of this practice session is to create and configurate the messages for an activity.
   

Start up JClic author, go to the menu File | Open File and choose the project module1.jclic.zip, as you have done in previous practice sessions.

Click Open.

Go to the menu Activities and in the column on the left select the activity Info.

Click on the menu Messages.

In this menu the three types of messages that can be in an activity are shown. In this session you will only put the initial message of the activity, but the procedure for the other two is identical.

The characteristics of the different types of messages are explained in the document Messages in this first module.

Tick the box corresponding to Inicial Message.

Click on the lower grey space which is lighter since you ticked the box.

In the window Box Content, you can write in the message.This can be a text message, an image, active content or a combination of all three.

Write the text message in the text box: Practice module 1, and click on the button Style . The style window will open.

In this window you can format the message. Try out the different possibilities, confirming with Ok to see the results.

Finally you configurate the style of the message in the following way:

Change the format of the text using Type, Size and Colour of text.

Tick the box Shadow and establish the colour from Colour of Shadow.

Change the Background Colour of the box.

When the values of the window Style are the same as the ones in the image confirm with OK.

In the window Box Content, click on the button . Select the folder course2.gif from the list of images in the media library and confirm the operation.

You will see that both the text and the image will be centered and for this reason are superimposed.To avoid this, click on the message box and tick the option Avoid Superimposing image/text. Confirm with OK.

Now you only have to situate the image and the text.Use the buttons with arrows corresponding to the text and the image to place the text in the centre and the image to the right.

Verify that the activity works in the test window using the button .

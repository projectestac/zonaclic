Activity options
The first of the tabs which are found in Activities is Options. This tab is common to all the types of JClic activities.

From this tab you can define some of the characteristics which have already been seen in the tab Project. The difference is that the changes made in this tab only affect the activity that is being edited.

These characteristics are:

  • Description

    Informs you of the type and name of the activity and allows you to make changes if you wish. The type changes are only possible between similar types of activities.

Also contains a box where you can write a description of the activity.
  • Reports

    Allows you to decide if or not you wish the results of the activity to be included in the user reports.It is advisable to leave this box unticked in those activities which are merely informative or that have some type of content irrelevant to the global evaluation of the project.

 
  • User interface

    From here you can define the skin for the activity you are editing.This skin will take preference over the project skin is there is one

    You can also find the button which allows you to put different sounds of events to the pre-defined ones in the project.
  • Content generator

    The activities can use automatic content generators, which are auxiliary programmes in charge of defining the content of the activity that will finally be shown to the students.At the moment of writing this document, there was only one content generator available called Arith which supplies the activity for mental arithmetic with exercises based on the criterias at the moment of designing the activity

    This theme will be treated in more detail in the document Automatic Content Generators in module 3.

 
  • Counters

    Another characteristic of the activities that can be determined from Options is to show or hide the time, try again or correct guess counters.You can also establish a maximum time limit or maximum number of tries for the activity.

 
  • Buttons

    The help or information buttons can be hidden or on view in the activities.

    From here the help content and type of information are established.
  • Shuffle

    The number of times that pieces can be shuffled around are determined here. This value is only used in activities where the information is mixed up.

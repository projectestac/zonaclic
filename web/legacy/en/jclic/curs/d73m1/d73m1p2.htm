Configuring JClic
 
The objective of this practice session is to see how to configurate some basic options of JClic e. g. protect with a password and change the aspect and skin.
   

 

Open JClic using the Start menu | JClic or using your desktop icon . If you still haven´t got this icon or have accidentely deleted it, use the Java Webstart Applications Manager. Go to the menu See-Downloaded Applications , select the icon JClic activate the menu Application - Create Direct Access (shortcuts or abbreviated methods depending on the version).

If this option does not appear return to the menu Application and select the option Eliminate Direct Access . Return to the menu Application and you will see that the option Create Direct Access has appeared.

If you have any doubts on how to create or eliminate shortcuts, consult the FAQ of clicZone.

With JClic open go to the menu Tools | Configuration.

In the window that has opened go to Security.

Select the option Protect changes of configuration with a password. Now you can introduce your password.

In the box New password write jclic (this is your password for the activity).In the box Confirm password write jclic again.

Confirm with OK

Return to the menu Tools | Configuration This time the window will not open directly but will ask you for the password to enter.

Introduce the password (jclic) and confirm with OK . Now you can access the configuration window.

This option stops unauthorised people from changing the configuration of JClic. It also protects the library in as much as no unauthorised person can add new projects or modify the icons which serve to activate them.

Now deactivate the password.To do so unmark the box Protect changes of configuration with password and confirm with OK. Now you can freely access the configuration and modify the project library.

Return to the menu Tools | Configuration and select the menu General.

In the dropdown menu Aspect select system.

In the dropdown menu Skin select @simple.xml

Leave the language as it is . If you wish to change the language use the dropdown menu Language and select the one you want. It must be taken into account that although the list is very long (it corresponds to the list defined in the standard ISO-639) the majority of languages are not available as yet..

Confirm the changes with the button OK.

You can try out different aspects and skins and choose the one you prefer. The changes you make will not affect the activities, only the way you see them.

The aspect that you can see in the images in this course correspond to the modal system working in Windows 2000 and Java 1.4.2.

Creating a new activity
   

The objective of this practice session is to learn how to make a new activity for a project.

First you will learn how to use the media library to manage the resources which will be used for the activities during the course (images, sounds and animations).

Then we will do a simple activity: make an information screen.

To do this practice session you need the following files:course.gif and course2.gif from the course material. These are found in the folder \files.

It is advisable to have prepared a folder which contains the files you need to use for the activities, preferably the same one where you keep your project. However, during the practice sessions of this course you will look for them in other folders.

You can visualize and manage the resources used in the project in the media library. The first step is to add these resources which can be of various types; the most common are images, sound and animations in various formats.

Start up JClic author, go to the menu File | Open File and select the project module1.jclic.zip which you created in the previous practice session and which can be found in the folder C:\Programme Files\JClic\projects\module1 (in Windows) or in $home/JClic/projects/module1 (in other systems).

Confirm with Open.

Click on the tab Media Library. At the moment the library is empty.

Click on the button to add the images to the library. When the window opens, go to Resource Search... go to the folder where you can find the course files and select course.gif and course2.gif, holding down the Control key to select both at the same time.

Once you have selected them confirm with Open.

A message like this will appear due to the fact that the files are not in the project file:

Confirm with Yes.

This message appears twice because you previously selected two files which are not in the project folder. Confirm both times.

You now have the files in the media library. Now you can create the activity.

Go to the tab Activities and click on to add a new activity to the project.

From the list on the left of the window New Activity select Information Screen.

In the box Name at the bottom of the window write the name of the activity: info.


Confirm with OK.

Go to the tab Panel. This is where we give shape to the activity.

The information screen starts off with only one box. More can be added but this is not necessary at the moment.

Click on the button Image in the tab Panel.

In the window Selection of Multimedia Object, you can find a list of all the images in the media library. Select course.gif.

Confirm with OK.

The size of the panel automatically adapts to the image as we have indicated an image for the whole panel not just a box.

If you did the same but clicking on the box and working from the window content you would have to adjust the size of the image manually to get the correct dimensions.

Check that the activity works by clicking on .

Close the test window and save the project with the menu File | Save..., select module1.jclic.zip and confirm with Save. Confirm again when asked if you wish to replace the file.

You will use the same file for the following two practice sessions.

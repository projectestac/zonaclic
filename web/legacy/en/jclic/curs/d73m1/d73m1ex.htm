The Creation of Educational Activities using JClic
Exercises
Creating a project with an information screen
   

Create a new project as you did before in practice session 3, with a new activity of the same type as in practice session 4, i.e. an information screen.To do this you will need the files computer.jpg and logo.gif which can be found in the folder with the course materials.

Once you have created the activity you must modify all the aspects necessary to make it as similar as possible to the one in the picture.

You must work with the four menus of the activity.

Aspects to be taken into account:

user interface

counters

the aspect of the windows (background colour, gradients, margin, position, edging)

the style of the messages (background, font, size, colour, shading, image, position of the elements)

the style for the edging round the panel (width and colour)

When you finish the exercise, save it and send it to your tutor.

 

You can see the example by clicking on the image.

Adapting an existent project and making an applet
   

Look for the project "The Balearic Islands" in the project library in the clicZone, created by Àngela Martín and Joan Saura from the school "Montaura de Manacor de la Vall" .

Look at the project content: you will see that there are activities with generic content (localization of the islands, their silhouette on a map, mountain chains in Mallorca...) and other content of a more detailed nature which would not be worked on so much with children from other parts of Spain or abroad: localization of all the townships of each of the islands, name of mountains etc...

The exercise we suggest consists in editing the project using JClic author and adapting it so as to use it with your pupils. Starting with the tab Sequences, eliminate the elements you do not wish to include. (To eliminate elements in the sequence use the button which shows a dustbin). Then go to the tab Activities and eliminate the same activities which you have just eliminated from the sequence, using the same button as before.

If you wish you can also make modifications in some of the activities (font, colours, fix a puzzle with two identical boxes, play around with the shape generator...) or add an activity you consider to be interesting or useful.

Modify the image on the title page so as your name appears. To do this you must:

  • Look for and select the image port-bal.gif in the media library.
  • Click on the button Save file (extract from ZIP) which appears at the bottom, next to the name of the file, and click OK when you are asked if you want to save.
  • With a graphic´s editor (e.g. paintShop Pro or Gimp) open the file port-bal.gif you will find in the folder C:\Programme files\JClic\projects\baleares. Modify it by adding your name as the author of the adaption and save it under the same name and in the same place.

  • Return to JClic author and click on the button Update resource content . This will cause the programme to refresh the image with the new changes. .
  • Try out the activity.

Put a note into Revisions in the tab Project, indicating the reason for making the new version. Finally, after changing the original name to baleares2.jclic.zip save the project. As you save the project, JClic author will ask you if you wish to eliminate the elements in the media library that will no longer be used. Your reply should be Affirmative..

You can put this exercise on your practice web in an applet as you saw in the document The JClic applet and in practice session 3.

Sequencing activities
   

 

 

The sequences in the JClic projects are lists where the order in which the activities must be shown to the pupils is specified, and the function to be given to the forward and back buttons each time.

It is important to sequence the activities properly, trying to progress through levels of difficulty and to combine the explanatory elements with the purely evaluative ones, giving the whole coherency.

Going from one point to another in the sequence can be done in three ways:

  • Automatically, when a certain amount of time has gone by after finalizing an activity.
  • By clicking on a box which has as its active content the ability to jump to a certain point in the sequence.This technique is used especially in activities which act as menus in projects which are made up of different sections.
  • By clicking on the JClic forward or back buttons to move in one direction or the other.

The use of the forward and back buttons are configured, for each element on the list, from the options panel on the right:

In a sequence you can mark certain elements with a "label". These labels are like bookmarks which help us to define jumps from one place to another.

For example, in a packet which contains science we can mark the first activity that refers to social science with the label "social sciences", another label with "natural sciences" in the first activity for natural science and a third label "start " at the first point of the sequence. This helps us to programme the jumps from the boxes in the initial menu screen to the points in the sequence where each type of activity begins, and also configure the forward and back buttons to allow us to return to the initial menu as we finish each series:

Firstly we prepare the activities and create a sequence listing all of them.The buttons on the tool bar will help us to add to or eliminate elements from the list as well as modifying their order.

 

Next we label important points in the sequence.To define a label you must select the element and write it in the text control on the right:

 

The next step is to adjust the working of the forward and back arrows. The possible actions are: Forward (or back), Pause (the button becomes inactive), Jump to a label, Return to the place where the last jump took place from, or Exit JClic:

In our case we have to make the following modifications:

Forward Button (forward arrow):

  • We deactivate this in "options menu ". In this activity it is obligatory to select an option as there is no sense in continuing going forward using the forward arrow.
  • In the final activities for the social science and natural science blocks we assign the "Return" function to this button. This ensures that we will return to the activity "Options menu ". We could also assign the function to "Jump to ..." adding the label "Start ", but it is preferable to give general orders ( e,g, back) instead of specific indications as this makes the projects more flexible.

Back Button (back arrow):

  • We desactivate this for the first point in the sequence as there is no point in going back. If we left the button activated JClic would make a linking circle and go back to the final activity on the list (Fauna).
  • We also deactivate it for the first elements in the social science block (story 1) and the natural science block (Natural Environment). In this way we avoid going from one block to another with the Back arrow.

You can see that JClic author draws red lines to define the limits between each block in the sequence.

JClic also allows the use of conditional jumps, which deviate from the path which should be followed, according to the results obtained and the time taken in doing the activities.With conditional jumps, packets of exercises can be prepared which increase or decrease the levels of difficulty according to the percentage of errors or correct answers given by each pupil when doing the activities.

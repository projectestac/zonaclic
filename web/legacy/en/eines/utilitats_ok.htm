Tisores Francesc Busquets

Have you found any time with which after compacting a file PCC it|he|she occupies more than it|he|she fits in a diskette and there is not way of taking it|him out of the computer where you work? This utility serves for cutting a big file in small bits, in order to power copying them in diskettes or sending them for e-mail when the recipient has the capacity of its|his|her|their mailbox limited. It is used for any type of file (PCCs, EXEs, WAVs...) and its|his|her|their functioning|operation is very simple: select the file which you want to cut in bits, indicate the desired size and the program will generate a series of files with extension|extent 001, 002, 003... It|He|She also generates a file BAT that is used for assembling the parts|pieces again when the original file has to be reconstructed. Once the parts|pieces assembled already the bits (001, 002...) can be erased and also the file BAT.

CoolEdit '96 Syntrillium

Very powerful editor|publisher of digital sound, with functions of recording, filters, reduction of noise, distortions, echoes...

The shareware version for Windows only allows using two groups of functions every time that it|she starts off. We recommend you to choose the first (Save, external clipboard...) and the fifth (Amplify, envelope, channel mixer and normalize).

A very practical function of CoolEdit is to amplify uniformly the volume of a file of digital sound in order to be able to listen to it at an optimum level. This function is called Normalize, and is in the menu Transform-Amplitude.

CoolEdit is used also for generating files of sound in compressed format|layout WAV. To put a file away in this formed it|he|she has to be done File - Ace Save, to select the type of files ACM Waveform (WAV), to do click in Options and Layer -3 to look for the format|layout MPEG.

Audacity
Free software under license GPL

Audacity is an editor|publisher of audio digital multitracks, with functions of recording, reproduction, import and export to different formats|layouts, effects VST and frequency analyzer. It is cross-platform, with versions for Windows, Linux and Mac OS X.
Editor|Publisher Image Microsoft

This program allows creating files ICO that can be used as icons for your packets of activities Click 3.0 in the installers created with the utility MKInst. The icons have to be created in EGA/VGA format|layout and always have a dimension of 32 x 32 and 16 colors.

When drawing an icon take that the points painted with the color "screen" will be transparent (the color that the user has selected for the background|bottom|fund of the screen will be seen) and those that have the color "inverse" will be seen with an inverse color to that of the screen of the user into account.

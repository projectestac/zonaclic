Instal·lació del motor Java

El JClic és una aplicació feta amb Java. El Java és una eina de desenvolupament que permet crear aplicacions que poden funcionar en diversos tipus d'ordinadors i sistemes operatius. Per crear-les cal disposar d'un programari especial, anomenat màquina virtual Java. El JClic necessita una màquina virtual Java versió 1.7 o posterior.

Un cop creades, les activitats es poden exportar a HTML5 per ser utilitzades en qualsevol dispositiu (ordinador, tauleta, mòbil...) que tingui un navegador web modern. El Java només és necessari en el moment de crear o modificar les activitats.

Per instal·lar el Java visiteu el lloc web https://java.com/es, feu clic al botó vermell i seguiu les instruccions.

Als entorns Windows hi ha dues versions de Java disponibles: la "normal" de 32 bits, identificada com a "x86", i una "especial" de 64 bits identificada com a "x86_64". Es poden tenir les dues variants instal·lades alhora.

Per comprovar quina versió teniu instal·lada aneu al Tauler de control, feu clic a la icona Java, pestanya Java, botó Veure... Allí hi ha una columna on es mostra l'arquitectura. Si hi teniu alguna "x86" no cal que feu res més, però si hi ha només "x86_64" serà recomanable afegir també la de 32 bits, que podeu descarregar des d'aquesta pàgina.

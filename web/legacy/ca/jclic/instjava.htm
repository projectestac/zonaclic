Instal·lació del motor Java

El JClic és una aplicació feta amb Java. El Java és una eina de desenvolupament que permet crear aplicacions que poden funcionar en diversos tipus d'ordinadors, sistemes operatius i navegadors. Per executar-les cal disposar d'un programari especial, anomenat màquina virtual Java. El JClic necessita una màquina virtual Java versió 1.5 o posterior.

A més de la màquina virtual, convé tenir instal·lats dos elements més: el sistema WebStart, que facilita la descàrrega, instal·lació i actualització de les aplicacions, i una extensió multimèdia que permeti visualitzar els vídeos i altres continguts especials de les aplicacions.

En aquesta pàgina trobareu informació per a instal·lar i configurar aquests components en:

Windows
Mac OS X
Linux / Solaris

Instal·lació en sistemes Windows

 

Abans de començar: Al Windows XP i 2000 només els usuaris amb perfil d'administrador poden instal·lar o actualitzar components del sistema. La majoria d'errors i problemes es donen quan la instal·lació s'intenta fer des d'una sessió iniciada per un usuari sense drets d'administració.

Per saber si ja teniu el Java instal·lat aneu a Inicia - Configuració - Tauler de control i busqueu-hi una icona semblant a aquesta:

  • Si hi és, obriu-la i comproveu la indicació de versió que apareix a la pestanya "Quant a" (About).
  • Si no hi és, o si és d'una versió anterior a la 1.5, us caldrà instal·lar el motor Java. Aneu a java.com i seguiu les instruccions que s'hi indiquen.

  • Sigui quina sigui la versió de Java que tingueu instal·lada, serà convenient afegir-li el complement Java Media Framework, que serveix per visualitzar els vídeos i altres elements multimèdia. Descarregueu i executeu aquest fitxer:

    customJMFinstal.exe (2.061 Kb)

Instal·lació en sistemes Linux / Solaris


En sistemes Linux i Solaris cal comprovar quuatre coses:

  1. Que el vostre sistema tingui instal·lada una màquina virtual Java

    La majoria de distribucions incorporen el paquet openJDK de sèrie. Comproveu amb el vostre gestor de paquets que aquest component estigui instal·lat.

    També podeu anar a java.com i descarregar la darrera versió per a Linux. Si us plau, llegiu atentament les instruccions d'instal·lació.

  2. Que el complement WebStart estigui instal·lat

    Les versions més recents de Java el porten incorporat per defecte, però en alguns casos cal instal·lar-lo com a paquet opcional.
  3. Que el navegador reconegui el Java

    Amb Mozilla o Firefox, escriviu a la barra d'adreces about:plugins i observeu si a la llista hi apareix el Java. Si no hi és caldrà crear un enllaç tou des de la carpeta plugins del navegador cap a un fitxer localitzat a la carpeta plugins del Java. A les FAQ del projecte Firefox s'explica com fer-ho.
  4. Instal·lar el complement Java Media Framework

    Aquest complement és necessari per veure les activitats que fan servir vídeos i altres continguts multimèdia. El podeu descarregar des de la web de Sun:

    http://java.sun.com/products/java-media/jmf/2.1.1/download.html

    És important llegir atentament les instruccions d'instal·lació!

Descàrrega i instal·lació del Clic 3.0

Totes les versions del Clic 3.0 funcionen en l'entorn Windows 3.1, 95, 98, Me, 2000 i XP. El JClic és una versió més actual del programa, que funciona en la plataforma Java.

Clic 3.0 no funciona ni amb Windows Vista ni amb Windows 7, per a aquests sistemes operatius és imprescindible descarregar JClic.

La distribució del programa és gratuïta per a usos educatius i no comercials.

L'instal·lador copia al disc dur el programa Clic, les utilitats ClicDB (gestió de la base de dades d'informes) i ClicPac (compactador / descompactador de paquets), l'ajuda i un paquet d'activitats d'exemple.

També es troba el manual d'ús del Clic que es distribueix en format PDF. Per visualitzar-los i/o imprimir-los es pot fer servir la utilitat gratuïta Adobe reader.

El Clic 3.0 està disponible en set idiomes:

català | espanyol | basc | gallec | anglès | francès | alemany

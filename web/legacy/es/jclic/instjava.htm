Instalación del motor Java

JClic es una aplicación realizada con Java. Java es una herramienta de desarrollo que permite crear aplicaciones que pueden funcionar en diversos tipos de ordenadores, sistemas operativos y navegadores. Para crearlas es necesario un software especial llamado máquina virtual Java. JClic necesita una máquina virtual Java versión 1.7 o posterior.

Una vez creadas, las actividades se pueden exportar a HTML5 para ser utilizadas en cualquier dispositivo (computador, tablet, móvil...) que cuente con un navegador web moderno. Java solo es necesario para crear o modificar actividades.

Para instalar Java visite el sitio web https://java.com/es, haga clic en el botón rojo y siga las instrucciones.

En entornos Windows hay dos versiones de Java disponibles: la "normal" de 32 bits, identificada como "x86", y una "especial" de 64 bits identificada como "x86_64". Se pueden tener las dos variantes instaladas al mismo tiempo.

Para comprobar qué versiones de Java tiene instaladas vaya al Panel de control, haga clic en el icono Java, pestaña Java, botón Ver.... Allí encontrará una columna donde se muestra la arquitectura. Si tiene alguna "x86" no es necesaria ninguna acción adicional, pero si hay solamente versiones "x86_64" será recomendable añadir también una de 32 bits que puede descargar desde esta página.

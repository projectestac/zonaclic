Los archivos ejecutables del JClic están firmados digitalmente con un certificados provinente provisto por la Ag�ncia Catalana de Certificaci�. En el caso que el motor Java de vuestro ordenador no reconozca como válida esta firma digital, será necesario importar manualmente el certificado raíz de la etidad emisora. En este documento se explica como hacerlo.

Los pasos que se descriven a continuación se deben ejecutar por cada uno de los usuarios del ordenador.

Antes de empezar, es necesario comprovar que el sistema esté utilizando una versión actualizada de Java.

  • Windows
  • Linux
  • Mac

Instruccions per a sistemes Windows

Paso 1: Descargar el certificado de la Ag�ncia Catalana de Certificaci�

Descargad el fichero clauEntitatCertificacioVinculadaSecretariaAdministracioFuncio.cer (1,92 KB) y guardadlo en una ubicación conocida. Por ejemplo, en el escritorio.

Paso 2: Abrir el panel de control de Java

Abrid el Panel de control i haced clic en el icono de Java.

Paso 3: Importar el certificado

  • Id a la pestaña Security i hadec clic en el botón Manage certificates...
  • En la ventana se muestran dos pestañas: "User" y "System". Dejad seleccionada la pestaña User.
  • IMPORTANTE: Desplegad la llista Certificate type y seleccionad el valor Signer CA
  • Haced clic en el botón Import

  • IMPORTANTE: En el diálogo de selección de ficheros, desplegad la lista Files of type y seleccionad el valor All files.

  • Buscad el fichero clauEntitatCertificacioVinculadaSecretariaAdministracioFuncio.cer que habéis descargado en el paso 1 (seguramente está en el escriptori) y confirmad.
  • Debería aparecer un elemento nuevo en la lista con los atributos "EC-SAFP" y "EC-GENCAT"
  • Confirmadlo todo y cerrad el panel de control de Java.

A partir de aquí ya deberíais poder ejecutar sin problemas los applets y aplicaciones Java WebStart firmadas por el Departament d'Ensenyament. La primera vez que lo hagáis os aparecerá una ventana de diálogo como esta:

Si marcáis la opción Do not show this again for apps from publisher and location above y hacéis clic en Run, ya no aparecerá más.

Instruccions per a sistemes GNU/Linux

La majoria de sistemes Linux (incloent la Linkat) utilitzen un motor Java anomenat OpenJDK, tot i que també és possible instal·lar-hi el Java d'Oracle. En aquest document es combinen les explicacions per als dos sistemes.

Pas 1: Descarregar el certificat de l'Ag�ncia Catalana de Certificaci�

Heu de descarregar el fitxer clauEntitatCertificacioVinculadaSecretariaAdministracioFuncio.cer (1,92 KB) i guardar-lo en una ubicació coneguda. Per exemple, a l'escriptori.

Pas 2: Engegar el tauler de control de Java

Executeu l'ordre "javaws -viewer". Si s'obre una finestra amb el t�tol "Java Cache Viewer", tanqueu-la. Nom�s ha de quedar una finestra amb el t�tol "Certificates" (si teniu OpenJDK) o "Java Control Panel" (si teniu el Java d'Oracle)

Pas 3: Importar el certificat

  • Només si teniu el Java d'Oracle: Aneu a la pestanya Security i feu clic al botó Manage certificates... (amb OpenJDK no cal que ho feu, perquè ja us trobareu directament a la finestra de certificats)
  • A la finestra es mostren dues pestanyes: "User" i "System". Heu de deixar seleccionada la pestanya User.
  • IMPORTANT: Desplegueu la llista Certificate type i seleccioneu el valor Trusted Root CA Certificates (si teniu OpenJDK) o Signer CA (si teniu el Java d'Oracle)
  • Feu clic al botó Import
  • IMPORTANT: Al diàleg de selecció de fitxers, desplegueu la llista Files of type i seleccioneu el valor All files.
  • Busqueu el fitxer clauEntitatCertificacioVinculadaSecretariaAdministracioFuncio.cer que heu descarregat al pas 1 (segurament és a l'escriptori) i confirmeu.
  • Hauria d'aparèixer un element nou a la llista amb els atributs "EC-SAFP" i "EC-GENCAT"
  • Confirmeu-ho tot i tanqueu el tauler de control de Java.

A partir d'aquí ja hauríeu de poder executar sense problemes els applets i aplicacions Java WebStart signades pel Departament d'Ensenyament. La primera vegada que ho feu us apareixerà una finestra de diàleg com aquesta:

Si marqueu l'opció Do not show this again for apps from publisher and location above i feu clic a Run, ja no us apareixerà més.

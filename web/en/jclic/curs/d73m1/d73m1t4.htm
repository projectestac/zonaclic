Types of Activities

JClic allows you to make seven basic types of activities:
  • Association games .The user must discover the relationship between two groups of information.
  • Memory games (pelmanism). The user must discover hidden pairs of elements the same or related to each.
  • Exploring, Identifying and Information which start off with one group of facts.
  • Puzzles,where out of order information has to be put in order. This information can be graphic, text, or sound ....or combine graphics and sound at the same time.
  • The activities with written answers which are resolved by writing a text which can be one word or a sentence.
  • Text Activities, which are exercises based on words, sentences, letters and paragraphs in a text which must be completed, understood,corrected and put in order. The texts can also contain images and active content windows.
  • Wordsearches and crosswords are interactive variants of these wellknown pastimes.

Some of these types have variants which gives us 16 different possibilities:
Types DESCRIPTION
Association Simple There are two groups of information with the same number of elements.Each element of the original group has a counterpart in the second group.
Complex There are also two groups of information, but these can contain a different number of elements and between these there can be different types of relationships: one to one, various to one, loose elements...
Memory

This type of activity consists of discovering pairs of elements hidden in boxes. The pairs may be formed by two identical pieces or two elements related to each other.In each go two pieces are uncovered which " hide" again if they do not form a pair. The objective is to find all the pairs in the panel.
Exploring Activity You are shown an initial piece of information and when you click on it you are shown another piece of information depending on the element you chose.
Identification Activity You are given one piece of information and you must click on those elements which fulfill certain conditions.
Information screen Information is shown and as an option you are offered the chance to activate the multimedia content which each element contains.
Puzzle Double Two panels are shown. In one the information is mixed up and the other is empty.You have to reconstruct the object in the empty panel by taking the pieces from the other panel one by one.
Interchangeable In only one panel the information is mixed up.In each try two pieces are moved around until the information is in the correct order.
Hole In the only panel available you make the pieces disappear one by one and each time a piece disappears the information is mixed up again. In each go,you can move one piece towards the hole until all the pieces are in the original order.
Text Complete the text You eliminate certain parts of a text ( letters, words, punctuation signs, sentences ) and the user has to fill them in.
Filling in spaces Certain words letters and sentences are selected from a text and hidden or camouflaged, and the user must fill them in.There are different ways of solving this: writing in an empty space, correcting the mistakes in an expression or by selecting answers from a list.
Identifying elements The user has to click on certain words, letters, numbers, symbols or punctuation signs.
Putting elements in order At the moment of designing the activity some words or paragraphs are mixed up and the user has to put them in the right order.
Written replies Information is shown and for each of the elements the corresponding text must be written.
Crosswords The user must fill in words in the panel according to the definition for each. The definitions can be written, graphic or sound. The programme automatically shows the definitions of the two words that cross where the cursor is found at any given time.
Wordsearch

Hidden words have to found in a panel of letters which are selected at random in each game.

The game may be associated to other content in which case, pieces of information (text, sound, images or animations) will appear each time a new word is discovered.

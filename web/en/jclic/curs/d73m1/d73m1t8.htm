The Media Library

The media library is the warehouse for the multimedia resources in a Project.

You can manage the images and other resources that are useful to the project from the media library.

In the library's main window you can see miniatures of the images and the animated gif. The remaining materials are represented by an icon which indicates the type of resource.

From the library you can add, erase, or simply visualise the multimedia resources for the project by using the buttons at the top.

Add an image or a multimedia object to the library.

Show a preview of the resource.

Eliminate the selected resource if it is not being used in an activity.The erase button will not be active if the resource is being used in any of the activities in the project.
Updates all the resources and returns them to the media library with the recent modifications.
Extracts all the resources from the file jclic.zip and sends them to the project folder so as they can be edited for changes.

When resources are added you can decide if you wish to see them all or only ones of a certain type. This facilitates the task when you work with many files.

At the bottom of the window you can find information on the properties of the resources,e.g. type of file, name, size and in which activity or activities they are being used. You also find the buttons to change the name, or the file, for another, and to extract and update the selected file.

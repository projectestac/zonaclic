Demonstration activities and new features of JClic
   

The best way of learning about JClic is to examine the example activities.

In the clicZone there are links which allow you to see these activities in an applet or download and install them onto your computer.

Once downloaded, from JClic you can initiate the example activities by clicking on the demo button.

On the initial screen you are offered the choice between taking a trip through the different types of activities in JClic or go through the novelties JClic offers you with respect to the anterior version Clic 3.0 .

The first option shows examples of the different types of activities. It allows you to see and understand the various types of exercises you can make with the programme.

The second option shows examples of possibilities which Clic 3.0 did not offer but which are available with JClic.

These novelties are the following:

  • The use of JPG, PNG, GIFs animations and Flash films. The types of files that could be used with Clic 3.0 were very limited.With JClic the possibilities are far more and multimedia elements such as sound files and video in different formats can be used.
  • Shape generators . Up to now all the puzzles in the Clic activities were formed by rectangular pieces.With JClic the pieces can be of different shapes: polygons, elliptical......
  • Graphic environments. JClic allows the use of different user environments which act like "skins" or " disguises" for the programme. The user environments consist of graphic design of the windows, buttons and counters, and an XML document which describes its behavour and distribution on the screen.
    Each JClic project (or each activity in a project) can use a different environment.
  • Sounds of events .The programme incorporates some sounds which are activated by certain actions (like mistakes or successes). These sounds can be changed,or activated and deactivated in each activity or project.
  • Other new chaacteristics :

- Possibility of establishing a time limit for the activity.
- Activities which allow a maximum number of goes.
- Put the elements in an activity into a certain order.
- Memory games with two groups of information.
- Better visual effects.
- Embedding of TrueType fonts into the JClic projects
- Use of colour with channel alfa (semi-transparencies)

Creation of activities

Once the project has been created and put into the necessary files in the media library, you can begin to create the activity using the third menu in JClic author; Activities.

On the lefthand side of the JClic author window there is a column where you can see the list of all the activities in the project. Here is where you can select the activity you wish to modify once it has been created.

This menu contains a series of buttons and four more sections (tabs) with the pertinent buttons for the creation and/or modification of the activities.

Buttons
Add a new activity to the project.

Tries out the selected activity by opening a window similar to JClic´s.
This window must be closed before continuing work in JClic author.

Moves the selected activity up or down in the list of activities.
Copies the selected activity.
Cuts the selected activity.
Sticks the selected activity.
Eliminates the selected activity.
Copies some attributes of the actual activity to another by opening a window which allows you to choose which attributes you want to be copied and to which activity.
Menus
Contains the tools to configurate different options in the activity such as the title, author, skin, buttons...
Contains the tools for establishing the aspect, size and position of the window where the activity takes place.
The content and form of the messages in the activity are established here.
These three menus are common to all the types of activities.
/

The final menu has different tools depending on the type of activity that is being edited. They can be either Panel or Text.

Messages

The JClic activities can contain three types of messages:

  • an initial message, which appears when the activity commences and very often gives information on what to do.
  • a final message which is shown only when the activity has been solved.
  • a message of error, which can appear in the activities which have a time limit or a limited number of goes.

These messages can contain text, images, sound, animations or a combination of these resources.

From the menu Messages in the activity and which is the same for all the activities, the type of message to appear and its content are established.

To edit a message first the messages must be marked for the activity.

Once activated, the space of each of the activities acts like a box and by clicking on this, the window Box Content opens, from where the corresponding content is established.

This window is the one which appears when a random box is clicked on (message box or panels). It contains the tools for introducing the text and deciding the style (type of letter, colour, size, shadow...), choosing an image, establishing a colour or background gradient or an active content.

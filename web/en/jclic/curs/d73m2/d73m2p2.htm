Creating an interchangeable text puzzle.

The objective of this practice session is to make a puzzle in which the box content is text and can be changed around and put in order to make a sentence , in this case a definition.

Start up JClic author, go to the menu File | Open file and choose the project module 2.jclic.zip which is found in C:\Programme Files\JClic\projects\module2.

Confirm with Open.

Go to the menu Activities and click on the button to add a new activity to the project.

In the list of types of activities select exchange puzzle. In the box Name write the name of the activity: puzzle2. Confirm with OK.

Go to the menu Panel.

Modify the number of rows and columns that you want the puzzle to have and the width and height of both, giving the following values:

Remember that you do this by raising or lowering the numbers with the buttons +/- or by selecting the numbers and writing them directly into the boxes.

The boxes can also be enlarged by placing the mouse at the edge of the panel and when a double arrow appears, dragging the edge to the desired size.

Now go to the panel and click on the first box to start introducing the text which will form the puzzle.

In the window Box Content, click in the space to write the first part of the text for the puzzle: Omnivorous: and confirm with OK.

Do the same in each of the 6 boxes in the puzzle, writing in each box the part of the text that corresponds. The text you will write is the following:

Omnivorous: / Animal which / eats materials / that are of / animal and vegetable/ origin.

Click on the button Style in the panel and shape the text: letter type size 16 Verdana blue (RGB: 51, 0, 153).

Establish green as the background colour for the boxes. (RGB: 204, 255, 153)

Confirm with OK.

These changes, made from the style button on the panel, affect all the boxes.

Click on the first box to give this one a different format to the others.

By using the button Style in the content window of the box you can access the tools to modify the style of the chosen box. Change the text to black Verdana 17, heavy type, and the background of the box to orange (RGB: 255, 153, 102).

Untick the box Border so as the separation between the edges of the puzzle are not visible.

Go to the menu Messages and write and configure an initial message: Put the definition in order.

Now the final message: Correct!

Making a memory game

The objective of this practice session is to create a memory game made up of pairs of different elements of different types: animated gifs and text.
   

To do this practice session you need the following files: one.gif, two.gif, three.gif, four.gif, five.gif, six.gif, seven.gif, eight.gif, nine.gif and ten.gif of the course material.

Start up JClic author, and open the project module2.jclic.zip.

Go to the menu Media Library and the button Add the resources one.gif, two.gif, three.gif, four.gif, five.gif, six.gif, seven.gif, eight.gif, nine.gif, ten.gif. You can add them all at once using Upper case or Control.

Go to the menu Activities, add a new activity to the project, memory game, and call it memory.

Go to the menu Panel.

Click on the first box and once the window of content opens, click on the button Image .

Select the image one.gif from the library and confirm with OK.

Go through the same procedure for each of the boxes using the files two.gif, three.gif, four.gif, five.gif, six.gif, seven.gif, eight.gif, nine.gif, ten.gif

Tick the option Alternative Content which is at the top on the right of the panel and click on ALT to introduce the second elements for the pairs.

You now again have empty boxes in the panel.Click on the first one and introduce the text. In this case write: uno. Confirm with OK.

Follow the same procedure for all the boxes in the panel.You are going to write the numbers 1-10 in Spanish:

uno / dos / tres / cuatro / cinco / seis / siete / ocho / nueve / diez

Remember you have to write the numbers in the boxes that correspond to the images you put in them before.

Once you have filled in the panel with all the alternative content, click on the button Style and make the background of the boxes black. Make the letters white with an orange shadow and use the font Comic Sans MS, size 28.

Using this button and not doing the changes from the boxes saves time because the properties apply to all the boxes at the same time.

The panels should look like this :
Panel with ALT unticked
Panel with ALT ticked

Write the messages for the activity, e.g.Find the numbers / OK, and format them choosing the colour for the letters, the background, font and size.

Change the colour of the windows to make them similar to the image.

Aspect of window once the activity is solved.

Try out the activity.

Close the test window and save the project.

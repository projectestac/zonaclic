Change one type of activity for another
The objective of this practice session is to learn the procedure for changing the form of a previous activity, in this case an interchangeable puzzle for another type of puzzle.
   

Start up JClic author and open the project module2.jclic.zip.

Go to the menu Activities and, from the column on the left, select the activity puzzle2, which you created in practice session 2 of this module.

Go to the menu Options. The first thing you find is the type of activity, in this case puzzles.ExchangePuzzle, which corresponds to the module Java that controls the interchangeable puzzles.

Once you have created the activity you may wish to change the modal or simply want to try out different types before deciding on one in particular.

Click on the button which is next to the type of activity to change the type.

A window will open where you can see the list of the different types of activities that you can use to automatically change the activity.

Make sure you read the warning at the top of the window. You must be very careful when using this as by passing from one type of activity to another you may lose part of the activity which you are editing. You may also find after changing the activity that it doesn´t function correctly or does not work at all.

Change the activity puzzle2 for another type of puzzle, e.g., select Hole Puzzle.

Try out the activity in the test window using the button .

Close the test window to return to the author.

Make changes to other types of activities. Try out each one to see the changes.

You will find that there are no problems if you change the activity puzzle2, which is an interchangeable puzzle, to a Hole puzzle or a Double puzzle. You can also change this activity into an information screen. With other changes however, the activity does not work properly or you have to make modifications in the panel.

Close the test window and JClic author, without saving the changes. When the programme asks you if you want to save the changes click on NO.

Complet a text
   
In this practice session you will create an activity for complet a text which is possibly one of the most complicated to solve, as no apparent visual signs appear to indicate where a word or part of a word is missing. For this reason, you will do the practice session with auditory help. In this case, you can find the sound file in the course Files. You could also practise with the programme for the treatment of sounds and make the recording yourself.
   

Open the project module4.jclic.zip, create a Text: complete text activity and call it Complet text.

From this page of the BBC website, copy part of the following synopsis in the body of the text of the activity:

SYNOPSIS:

Relations between Canadian seal hunters and the animal rights groups that monitor their activities have never been cordial, but protestors with the Humane Society of the United States say they were shocked when a sealing boat rammed into their small inflatable dinghy. The craft's propeller was damaged but no-one was seriously hurt. But the incident comes just a day after infuriated hunters flung seal guts at the protestors who were crammed into the same boat.

A Humane Society spokesperson said in a statement that the sealers need to remember that the groups are legally allowed to be near the hunters as long as they don't directly interfere with the seal cull. But the sealers are also frustrated because of a huge fall in the number of seals they are taking in comparison to previous years. Warm weather has melted the ice floes, making their task much more difficult. And some hunters have tied up their boats, saying that many of the seals are too young to be legally killed.

You must add the title. Place the cursor in front of the first word (Relations...) and press Enter. Go to the top and click on the button . Fill in the dialogue box as you see in the following example:

Remember that the cell where you have introduced the title must be resized from the dialogue box (marked in red).

This activity is accompanied by a voice reading the synopsis.

Go to the Media Library and add the file seals.ogg which you have in the folder Files (this type of file was worked on in the introduction module 2. It can be reproduced using the programme Audacity).

Go back to the tab Text Content. Click on the button . In the previous screen, choose the following options:

The style is the same as you have chosen for the title box but with the colours the other way around. So as it becomes a single window, you can activate the main window (tab Window) and choose the same background colour (and the game window with the option Transparent activated).

Return to the dialogue box Previous window... Now you can introduce the voice reading:

First click on the box Previous message and immediately on the Active content button. Select Type: Play sound. Look for the file seals.ogg with the button . Press OK. you have to apply is the dark green background colour that you have used before. Then write: Click on the screen when you have finished listening to the text. Accept OK until you return to the activity.

Now choose the Create target () Which are the words that will not be seen on the activity screen. Mark: relations, monitor, cordial, rammed into their small inflatable dinghy, the craft´s propeller, flung seal guts, crammed into the same boat, legally, don´t directly interfere with the seal cull, have tied up their boats. Choose a style () using a size which makes the reading easier.

Activate the button Check. In this way JClic will let the user know when not all the words have been written.

Write the Initial message and the Final message.

Test that the activity works and save the project.

   

In the title for the synopsis you can also insert images. To do this, you must first save an image which you have extracted from the webpage on your hard disk. Then you can add it to the title cell taking into account that the the size of this cell will have to be larger.

You could also include it in the preliminary window under the command to listen to the synopsis.

Identify elements
   

In the identification of elements the user has to click on certain words, letters, numbers, symbols or punctuation signs which were defined as unknown elements at the moment of designing the activity.

The evaluation of this type of activity always has to be postponed. When the user clicks on the correction button, the programme visually shows the letter or words that have been wrongly marked, and if necessary, informs him/her of the number of elements that still have to be found.

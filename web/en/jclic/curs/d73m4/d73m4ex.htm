The Creation of Educational Activities using JClic
Exercises
Create a project with text activities
   

As an exercise in this module you have to create a new project.

Look for six short articles in a newspaper (or fragments of articles), of about 100 to 150 words. With this material you have to make:

  • A text activity of a Cloze type (filling in spaces in a text). The pupil has to fill in four missing words. Use help windows for the unknown words if the text is difficult or the context does not help the pupil to deduce the word easily.
  • Another activity of the same type but where the four unknown words are in a list of options.
  • Yet another activity of the same type where the first letter of each of the four unknown words is already in the space provided. The pupil has to complete the word.
  • A text activity to identify words in this case, all the common nouns in the text.
  • An activity where the paragraphs have to be put in order. It is important that the text you use is made up of a series of short logical paragraphs.
  • An activity to put words in order and where ten words from the text are mixed up.

To find suitable texts you can use a normal newspaper or the electronic edition of one which appears on Internet.

If you use the electronic version you can also use some of the images which go with the articles to put in a box in the activity. Make sure that these are quite small so as not to distract the pupil from the main reason for the activity.

Written answer
   

This type of activity has a visual aspect similar to the exploring activities: all the content of panel A and one box from panel B is shown. This is where the user has to write the text answer he/she considers correct to answer the box that is open at that moment in panel A.

There must be at least one answer for each of the boxes in panel A, and the correct answers must be written in the assigned box in the grid in panel B. The answers in this file have to be written in the same order as the questions in panel A, as we did in the exploring activities:

The programme never shows the content of panel B: it only uses it to decide if the answers given by the user are correct.

If a box allows more than one correct answer, you have to indicate this by separating all the possibilities with vertical lines on the same line for the text. e.g.,

Bike|Bicycle|Cycle

... the programme will consider the three answers correct. You can get the vertical line by pressing the keys AltGr + 1.

To evaluate the answers, the programme does not take into account if the letters are upper or lower case but any accents (where appropriate) or punctuation marks must be written correctly.

Order elements
   

In these types of activities, the user has to put the elements, (words or paragraphs) which appear out of order in the text, the right way around.

To create this type of exercise, it is necessary to write a text and mark certain words or paragraphs as unknown elements. When the activity begins, the unknown elements get mixed up. Putting them back in order is always done by changing the positions of a pair of elements in a similar way to the interchange modality of the puzzles. Obviously, to create an exercise of this type, a minimum of two unknown factors have to be created otherwise it would be impossible to mix up words.

The evaluation of the exercises is always postponed.

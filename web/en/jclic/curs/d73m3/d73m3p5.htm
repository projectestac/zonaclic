Creating an information screen

   

In module 1 you created a blank information screen. In this practice session you will make a new one but at the same time learn about more options such as Active Content and Multimedia.
   

Recuperate the project module3.jclic.zip. From Activities, add a new one , on information screen, and call it Choose.

From the tab Panel, change to 2 rows and 1 column.

Write in each box: Simple Association, in one, and Complex Association, in the other. Modify the style of the panel until you have something similar to this:

Resize the grid until you can read the text in the boxes correctly.

Click on the box where you have written Simple Association. Activate the button Active Content. Select the button from the dialogue panel..

This button means that when you click on the box you will jump to one of the activities in the same project you are working on. As you have already designed the two activities Simple Association and Complex Association, this information screen will take you directly to them.

Click on the button which selects the activity.

In the dropdown menu choose, Simple Association and confirm with OK.

Repeat the operation with the Complex Association box, choosing the adequate activity.

Verify that the information screen is working correctly from the Test Player.
Add an initial message and save the project.
   

You have found out with the Test Player that you couldn´t return from the activity you went to from a box. This is because you still haven´t done the sequence of activities which allows you to go from one activity to another. You will practice sequences in module 5 .

In the dialogue box Active Content and Multimedia, you saw the button . This button acts similarly to the "jumps" which you will work on in practice session 2 module 5.

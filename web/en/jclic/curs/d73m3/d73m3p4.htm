Creating an identification activity

The objective of this practice session is to make an identification activity with animated images. When using animated images you have to be careful as they can distract the user from the real objective of the activity.

You will also include alternative content in some of the boxes, taking advantage of the same animated files together with the text content.

Remember that it is essential to write the initial message as this is the guideline for the activity.
   

In the folder Files you have the following animated files: earth.gif, saturn.gif, jupiter.gif and pluto.gif. Put them in the Media Library, in the same way as you have done in previous practice sessions.Once there you can see that the files earth.gif and pluto.gif are visible but the other two not. If you double click, you can see them. The two that you didn´t see are those with a static background. However, all four files have animated content.

From the tab Activities, and clicking on , add Identify cells activity and give it the same name: Identify cells.

As there are four images, divide the single panel into 4 boxes (4 x 1, 1 x 4 or 2 x 2).

Introduce a different image into each box. Follow this order: Jupiter, Saturn , the Earth and Pluto.

Use the Style button in the panel to give the images a black background and light colours to the text, border and alternative content.

Before continuing with the content of the panels, go to the tab Messages. Write in Initial Message: Which planets are not Saturn? This means that the user must click on the boxes for Jupiter, the Earth and Pluto. Write the final message: Saturn: the planet with rings.

In the boxes with these planets you must write the alternative content (i.e.,what the user will see when he/she clicks on the box with animation and this disappears):

Go back to the tab Panel. Activate the option Alternative Content and press ALT:

The images have disappeared. The alternative content can now be written in the boxes.This is the button for passing from the principal content to the alternative content.

Click on the first box where Jupiter is. Write: Jupiter: the biggest planet in the Solar System. You can add the same image. To do this activate the option Avoid Overlapping Image / Text and situate the text and picture to your satisfaction.

In Pluto write: Pluto: the furthest planet from the Sun. And in the Earth: The Earth: the planet we live on. Saturn has no alterative content because, if the box is clicked on the answer is incorrect.


Now go to the tab Relationships. Click on Jupiter, Pluto and the Earth. The background colour will be seen as the inverse.

Test the activity then put the style you prefer to it.

Save the activity in module3.jclic.zip.

There are many possibilities for this activity. The same content can be used to make variations by changing the inital message and the relationships.

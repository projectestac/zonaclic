Creating an exploring activity
With an environment very similar to the association activities, in this practice session you will make an exploring activity. The objective is to create an exploring exercise using combined multimedia objects..
   

From JClic author, go to the tab Media Library to get the necessary files for this activity. These are: mozart.mp3, beethoven.mp3, prokofiev.mp3, vivaldi.mp3, saint-saens.mp3, and clau.gif (image taken from http://www.edu365.com/primaria/muds/musica/mozart/index.htm).

Click on . With the help of the dropdown menu select the option Sounds (*.wav, *.mp3 ) to add the sound files (remember that you can hold down the Control key to select various files at the same time), and Images (*.GIF...) for the image file.

Click on and create an exploring activity. Call it Explore activity. As there are 5 music files, divide the columns and files in both Panel A and Panel B so as there are 5 boxes (1 x 5 or 5 x 1).

In Panel A, introduce the names of the composers MOZART, BEETHOVEN, VIVALDI, PROKOFIEV and SAINT-SAËNS. Use a common style for the whole grid. You can see that the option Shuffle is not activated. This is not essential in this type of activity.

In Panel B, put the same image (clef.gif) in all the boxes (by clicking on the box and going to the button Image | Selection of multimedia object | clef.gif) and in each one, the sound file that you have in the Media Library. As you know, the order in which you introduce the sound files in Panel B may be the same as in Panel A. In this case it is preferable to take this into account so as not to have to change the relationships later.

Go to the first box in Panel B. Select the image (if you haven´t already done so) and then click on the button Active Content. Choose the button Interpret sound and look for the file mozart.mp3. Confirm in the three open dialogue boxes.

Do the same for the remaining boxes.

When all the boxes have their content, go to the tab Relationships. Make sure that everything is correct. Pair together the boxes with no relationship:

To finish, choose an adequate style for the windows and write an initial message. (You do not have to write a final message because the activity has no end).

Save the project, overwriting the file module3.jclic.zip.

   

You can make exploring activities with a different number of boxes in each panel. Following the example in the practice session, you can choose fewer composers and more works by each of them.

As a complementary activity, you can look on Internet for a picture of each composer and put the images in the boxes in Panel B.

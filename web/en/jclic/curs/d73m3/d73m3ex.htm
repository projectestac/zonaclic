Module 3
The Creation of Educational Activities using JClic
Exercises
Create a project from an information screen
   

In this module you have to create a new project which contains five activities: a simple association, a complex association, an exploring activity, an identification activity and an information screen with 4 boxes that will serve as the initial menu. You must be able to jump to any of the activities by clicking on the boxes in this menu.

Start by making the initial information screen, and put in the four boxes which contain the signs corresponding to each type of activity. Then make up the four activities and finally establish the active content for each box in the information screen in order to be able to jump to the activities.

All the activities have to start with the same theme. You can choose any one you want but the activity panels must combine different resources: images, text and sound in the form of WAV, e.g. MP3 or MIDI (in this case, the button Active Content is different to Interpret sound).

On the pages http://www.findsounds.com/ and http://www.wavcentral.com/ you can find several different types of sounds through a search engine in English. You can also find material in the sounds bank of the Ministry of Education.

The whole project must have the same skin (user interface) and you must make the aspect of the windows and the style of the messages coherent.

When you finish the exercise make sure you save it.

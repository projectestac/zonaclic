Automatic content generation

In this practice session you will make a simple mental arithmatic association using automatic content generation. If you want to use the model Arith with other types of activities, the edition of the content generator is applied in the same way.

This is the last activity for the project in this module. In module 5 you will reunite all the projects done up to now.

   

From JClic author create a simple association activity and call it Mental arithmetic.

In the tab Panel, select the lines and columns for each of the grids (A and B) as shown below. Give each grid its own style :

Next, go to the tab Options. Select the module Arith in the section Content Generator. Push the Configuration button.

Select or activate the different options for editing following this model:

Test that the association is working by previously extending the grids from the tab Panel. In this way you will be able to see the content more easily while you are testing.

From the test window you can clic on the button . You will see that each time the activity restarts with different operations.

Go back to tab Panel A in JClic author and activate the option Alternative content. Test the association once again. When you click on the correct answer, the full operation appears in the box.

The module Arith tries to generate operations at random from the established parameters if these are reasonable. If it finds unsolvable situations (e.g. sums with numbers between 10 and 20 and a result of less than 5) it finds itself obliged to infringe some of the conditions laid down.
